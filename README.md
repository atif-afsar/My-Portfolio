## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/atif-afsar/My-Portfolio.git
```

2. Install dependencies:
```bash
cd My-Portfolio
npm install
```

3. Start the development server:
```bash
npm run dev
```
